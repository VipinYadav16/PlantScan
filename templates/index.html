About the Project

This project leverages deep learning to detect plant diseases from leaf images, addressing significant agricultural challenges. Using the PlantVillage dataset, it implements two models: a custom lightweight CNN for efficient processing and MobileNet, a pre-trained model for higher accuracy. The solution is deployed through an interactive Flask-based web application, where users can upload leaf images to receive disease classifications along with confidence scores. This project demonstrates the potential of machine learning to enhance agricultural productivity and mitigate crop losses through accessible, real-time technology-driven solutions.
